CodeCounter:

To use this Add-In you must close all open vb's exept for the add-in ofcoarse.
compile it in your VB\Wizards dir (Eg. c:\Program Files\Microsoft Visual Studio\VB98\Wizards)
Open a new vb with a project. Make sure the add-in is startup/loaded in the add-in manager.
Choose "Code counter" from the add-in menu. 
And finaly: Tell all your friends, boss or co-workers how big your projects are to show off...

If you want to debug the add-in start CodeCounter.vbp
Set some breakpoints or something.
Click run. Start a new vb with a project.
Click on "Code counter" in the add-in menu.
